# Vendors

本仓库包含 NekokoLPA 2 支持的实体 eSIM 卡与读卡器的厂商数据。

**Language:** **中文** | [English](README.md) | [日本語](README_ja-jp.md)

## 结构

- `cards/`：按品牌分组的实体 eSIM 卡 YAML
- `readers/`：按品牌分组的读卡器 YAML

## 参与方式

1. 在 `cards/` 或 `readers/` 中新增/更新品牌 YAML。
2. 提交 Pull Request。

## 厂商说明

如果您是卡片或读卡器厂商并希望赞助本项目（哪怕是很小的 Aff 链接收益），请联系 `business@nekoko.ee`。

为您的卡片支持 NekokoLPA 2，请至少加入以下两个 ARA-M SHA1：

- `d1c0f48b370e74d4ea4770ed4c3cd70a3198d31f`
- `c47350c7ba682b34a3e584a0d58463ea42b1ad73`
